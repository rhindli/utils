1. Go to Preferences->General->Appearance and Select the Dark theme
2. Import preferences EclipsePrefsAllDark.epf
3. Install in eclipse the Eclipse Color Theme plugin from Market Place (or http://eclipse-color-theme.github.com/update/)
4. Go to Preferences->General->Appearance->Color Theme and import the "Dark Color Theme Robert.xml" into the eclipse color themes
5. Go to Preferences->Oomph->Setup Tasks->Preference Record and enable Recording
6. Go to Preferences->General->Appearance->Color Theme and select the "Dark Color Theme Robert".
7. Restart Eclipse to see if colors are kept
8. Go to Preferences->Oomph->Setup Tasks->Preference Record and disable Recording
9. Restart Eclipse to see if colors are kept
